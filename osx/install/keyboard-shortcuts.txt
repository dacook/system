There's no easy way to export these, just set these up manually. (Or start using a third party app)

Open up: System Preferences > Keyboard > Shortcuts.

```
Mission Control
    Move left a space           ctrl+cmd+left  // Not ctrl+left because I keep accidentally pressing it
    Move left a right           ctrl+cmd+right
    Switch to Desktop 1         ctrl+cmd+1  //and so on
    Show Dashboard              cmd+H // Because sometimes I accidentally press it, but I never want to instantly hide an app! So just do some other action.

Keyboard
    Move focus to the menu bar  ctrl+/
    Move focus to active or next window  alt+tab //sound familiar?? This is the same as alt+tab in windows! :D
    Move focus to status menus  ctrl+shift+/

Spotlight
    Show Finder search window   cmd+Q //Because sometimes I accidentally press it, but I never want to instantly quit an app! So just do some other action.

App Shortcuts  
    Mail.app
        Send                    cmd+return
        Paste and Match Style   cmd+shift+V
        Paste as Quotation      cmd+alt+V
    Google Chrome.app
        View Source             cmd+U //same as firefox
        Task Manager            alt+shift+esc //similar to windows (from memory)
        Matrix Keyword Helper   ctrl+w // using shift or cmd affects the way it's opened, and alt won't work
    Terminal.app
        Show Previous Tab       control+shift+tab  //or cmd+alt+left, or cmd+1
        Show Next Tab           control+tab        //or cmd+alt+right, or cmd+2
    Microsoft Word.app
        Document Map Pane       cmd+shift+D
        Paste and Match Formatting  cmd+shift+V
        Repeat $1               cmd+shift+z //hrmmm can't find a way to do this
    Firefox.app
        // shortcuts don't work :( https://bugzilla.mozilla.org/show_bug.cgi?id=429824 U
        // Instead, use https://addons.mozilla.org/en-GB/firefox/addon/s3menu-wizard/ (or start using a third party app)
    sublime
        Wrap at 120  ctrl+cmd+Q //would like to do this in sublime keymapping but don't know how to find the command name
